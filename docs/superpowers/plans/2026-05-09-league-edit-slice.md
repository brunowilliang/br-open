## Task 2: Add `update` and `delete` to League Management

**Files:**
- Modify: `convex/functions/league/management.ts`

- [ ] **Step 1: Add a reusable ownership lookup**

Keep the ownership check local to management.

Target helper:

```ts
async function getManagedLeagueOrThrow(
  ctx: AuthenticatedCtx,
  leagueId: Id<"league">
) {
  const currentLeague = await ctx.orm.query.league.findFirst({
    where: { id: leagueId, managerUserId: ctx.userId },
  });

  if (!currentLeague) {
    throw new CRPCError({
      code: "FORBIDDEN",
      message: "Liga não encontrada para esse gestor.",
    });
  }

  return currentLeague;
}
```

- [ ] **Step 2: Reuse the helper in `getById`**

Replace inline fetch with the helper so ownership logic stays consistent.

- [ ] **Step 3: Implement `update`**

Target mutation:

```ts
export const update = authMutation
  .input(UpdateLeagueSchema)
  .output(leagueSchema)
  .mutation(async ({ ctx, input }) => {
    const now = new Date();
    const leagueId = input.leagueId as Id<"league">;

    await getManagedLeagueOrThrow(ctx, leagueId);

    const [updatedLeague] = await ctx.orm
      .update(league)
      .set({
        name: input.name,
        description: input.description,
        regulation: input.regulation,
        city: input.city,
        state: input.state,
        locationNotes: input.locationNotes,
        visibility: input.visibility,
        categories: input.categories,
        coverStorageId: input.coverStorageId,
        avatarStorageId: input.avatarStorageId,
        updatedAt: now,
      })
      .where("id", "=", leagueId)
      .returning();

    return serializeLeague(updatedLeague);
  });
```

- [ ] **Step 4: Implement `delete`**

Target mutation:

```ts
export const remove = authMutation
  .input(DeleteLeagueSchema)
  .output(z.object({ success: z.literal(true) }))
  .mutation(async ({ ctx, input }) => {
    const leagueId = input.leagueId as Id<"league">;

    await getManagedLeagueOrThrow(ctx, leagueId);

    await ctx.orm.delete(league).where("id", "=", leagueId);

    return { success: true };
  });
```
```

Note:
- If `remove` conflicts with naming elsewhere, use `deleteLeague` or `deleteById`, but keep the API explicit.
